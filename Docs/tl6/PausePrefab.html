Description

This asset provides a ready-to-use pause menu for your Unity game. It eliminates the need to build a pause menu from scratch and helps you streamline your in-game UI experience.

The recording below displays the Pause Menu behavior and how it looks like in the context of the game "Breaking Red"

Components

  • Resume Game: Un-pauses the game and returns to gameplay.
  • Main Menu: Navigates the user back to the main menu screen.
  • Save Game: Triggers the game-saving functionality. (Requires a separate save system integration.)
  • Inventory: Opens the player’s inventory.
  • Quit Game: Exits the game application.

Setup Instructions

  • Connect a PauseMenu script to the Pause menu canvas prefab through the inspector. The script should include the behaviors for the following buttons: Resume, Main Menu, Save, Inventory, Quit Game.

Requirements

  • Unity 6000.0.38f1 or later.
  • Input: Unity Input System or legacy Input Manager.
  • Save and inventory system for full button functionality.

Usage

  • Press the assigned pause key to bring up the menu.
  • Menu UI is built with Unity’s Canvas, Panel, and Buttons for easy editing.
