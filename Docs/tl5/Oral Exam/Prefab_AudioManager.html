Description:

This asset provides a powerful and flexible Audio Manager system that allows you to add background music, environmental music, and sound effects. It supports fade in and fade out effects, pause or resume logic, and fully customizable settings like volume, pitch, loop, and mixer group.

Included in this asset:
  • AudioManager Prefab
  • AudioManager Class Script
  • AudioType Class Script (Custom class to store audio clip information, such as names, loop, volume, and so on)
Features:
  • Easy add unlimited audio clips with no repeated code. Using AudioManager.instance.Play("clipName") to play music and sound effects anytime.
  • Pause, Stop, or Resume audio by name
  • Loop support & Unity Mixer Group integration
Requirements: Unity 66000.0.38f1 or later
